### 连接方法
公网的那台用原来的方法就行了，内网的那两台无法直接访问，要用公网的主机做媒介，即：
A->B,B->C,D

最简单的方法是在公网主机用ssh访问，命令：
```
ssh -i "key.pem" ubuntu@10.0.0.12
```

```
ssh -i "key.pem" ubuntu@10.0.0.20
```
查看一下ip确认一下：
```
ifconfig -a
```
退出:
```
exit
```
另一种是本地用公网主机做跳板机直接访问（这个我大概查了一下，如果你有需要我就去弄，不过windows比较麻烦，要装ssh）
